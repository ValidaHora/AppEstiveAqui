Digite o código de 6 números que aparece na tela do PassClock no campo "Inserir Código" do app.

Pressione o botão "Lançar Hora" para lançar a hora.

Se você estiver com acesso à internet no momento da digitação, o código é enviado imediatamente para o seu gestor.

Caso você esteja sem internet, o código digitado é guardado no celular. Assim que você tiver novamente acesso à internet, os códigos são enviados para o gestor.
